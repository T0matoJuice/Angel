## 第一层：绝对指令（100%确定性）

### A类：直接判定为【质量工单】

1. **新机黄金法则**：购机/安装日期在30天内，且处理方案或来电内容包含"换机"、"退机"、"退货"
2. **产品鉴定政策**：服务项目为"产品鉴定"、"只换不修政策产品质量鉴定"，且处理方案包含"换机"、"退机"、"退货"
3. **核心部件更换**：旧件/新件名称不为空，且部件在【核心部件库】中
   - 核心部件库：电源适配器、微电脑电源板、微电脑显示板、控制板总成、主板、绕丝加热体总成、电热管、增压泵、电磁阀、进水阀、高压开关、TDS传感器、电控龙头、灯显龙头、浮球开关、浮球组件、流量计、温度感应器、滤网总成、指示灯板、排水接头、密封圈、真空热罐总成、抽水泵、跷板开关、反渗透膜滤芯、滤芯座总成
4. **滤芯质量缺陷**：旧件/新件包含"滤芯"，且来电内容或现场诊断包含"漏碳"、"黑点"、"黑渣"、"碳粉"、"活性炭粉末"
5. **保外转保内政策**：保内保外为"保外转保内"，且旧件/新件包含"滤芯"
6. **噪音换机**：处理方案或来电内容提及"噪音"或"分贝"，且结果为"换机"、"退机"、"退货"

### B类：直接判定为【非质量工单】

1. **外部加装**：服务项目包含"加装"（如：加装压力桶、加装排水盒、加装后置活性炭）
2. **安全维护**：服务项目包含"安全维护"
3. **用户/环境责任**：处理方案包含"用户"、"客户"、"台盆"、"厨房"、"下水"、"第三方"、"水压"、"水质"等，且描述为问题根源

### 学习任务

请从训练数据中学习以下规律：

#### 1. 字段权重体系
- **服务项目或故障现象**（权重40%）：
  - "质量鉴定"、"产品质量鉴定" → 质量工单（99%）
  - "安装"、"调试"、"加装"、"改装"、"移机" → 非质量工单（95%）
  - "维护"、"保养"、"清洗" → 非质量工单（80%）

- **判定依据**（权重30%）：
  - 产品缺陷（"零部件损坏"、"性能不达标"、"失效"）→ 质量工单
  - 环境/人为因素（"水压"、"水质"、"电压"、"操作不当"）→ 非质量工单

- **故障类别**（权重15%）：分析训练数据中的分布规律

- **部件更换**（权重10%）：旧件≠新件且都非空 → 提升质量怀疑权重+20%

- **辅助字段**（权重5%）：来电内容、现场诊断、处理方案

#### 2. 交叉验证规则
当第一层规则未命中时，使用以下逻辑：

- **功能性故障判断**：来电内容/现场诊断是否描述明确故障（漏水、不通电、不制水、不出水、不加热、噪音大、显示异常、错误码）？
- **维护行为判断**：处理方案是否为纯维护/复位（重启、复位、调试、清洗、紧固、重新安装、加固、指导、解释、检查正常）？

**决策逻辑**：
- 有故障 + 简单维护解决 → 非质量工单
- 有故障 + 非简单维护 → 质量工单
- 无故障 + 维护/解释 → 非质量工单
- 无故障 + 无维护 → 非质量工单

#### 3. 边界情况处理
- 信息不足：参考训练数据中相似案例的标注倾向
- 多因素混合：以主要原因为准
- 模糊表述：综合多字段判断
- 默认策略：信息严重不足时，默认为非质量工单
